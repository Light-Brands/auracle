# Round 7 — Top-to-Bottom Editorial Review (Vol 1: SEE)

A full editorial pass against the **current book manuscript**, run after Round 5
post-publication refinement, Round 6 Sage sensitivity refresh, the unified
woven integration (cultural + LGBTQ+ + male survivor weaving), and the
appendix migration to the companion website.

This round treats Vol 1 as the 17-chapter book it is now — not as the video
curriculum that framed Rounds 1–5. Earlier rounds asked questions that
presupposed a course product (lessons, decoder cards, modules). Round 7 reads
the prose as readers will encounter it: a single, sustained book.

## Scope

- **Source of truth:** `book/sovereignty-series/vol-1-see/vol-1-complete-manuscript.md`
  (15,948 lines, 17 chapters, plus front matter, vignettes, and back matter)
- **What's in:** book prose, chapter architecture, voice, pacing, prose craft,
  copy, proofreading, sensitivity (re-pass), fact-checking, reader experience.
- **What's out:** the companion website, course materials, marketing copy,
  workbook drafts, ePub/docx outputs (these are downstream artifacts).

## Key Differences From Prior Rounds

- **Book framing, not course framing.** Round 5 still asked about lessons,
  decoder cards as products, audio editions. Round 7 asks about chapters,
  sequencing, pacing, voice — book questions for a book.
- **Round 6 has landed.** Sage's sensitivity refresh decisions are
  implemented. Round 7's sensitivity pass (when it runs in Phase 2) is a
  validation read against current prose, not a re-litigation.
- **Cultural + LGBTQ+ + male survivor weaving is integrated.** The "Same wolf,
  different clothing" thesis is now in the chapters. Round 7 reads the result
  as a finished braid, not as separate threads.
- **Appendices are gone from the book.** Per the standing decision, hotlines /
  directories / URLs live on the companion website. Reviewers should not
  recommend reintroducing them into the printed book.

## How to Respond

For each question in a reviewer's issue file:

1. Select your preferred option (or write your own).
2. Add brief reasoning if the choice is non-obvious.
3. Answer any clarifying questions.

Author responses go in `phase-1-strategic/author-decisions.md` (created when
you start responding). Once Phase 1 decisions are recorded, mark the round
state to `phase_2_ready` to unblock Phase 2 reviewers.
