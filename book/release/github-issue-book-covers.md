# Designer Brief: Cover Design for The Sovereignty Series (8 Volumes + 8 Companion Workbooks)

## Overview

Design a cohesive cover system for **The Sovereignty Series** — an 8-volume nonfiction series about healing from narcissistic abuse and building personal sovereignty, plus **8 companion workbooks** (one per volume).

- **Author (on cover/spine/title page):** Jennifer Brooke Lawless *(the "M.S." appears only in the back-cover author bio)*
- **Trim Size:** 6" x 9"
- **Paper Type:** White (black ink interior)
- **Volumes:** 8 main + 8 companion workbooks = **16 covers total**

## Cover Layout Details

### Spine Text (Paperback)

Standard spine format for each volume:

```
SEE  |  Recognizing Narcissistic Manipulation in Relationships, Family, and Work  |  Jennifer Brooke Lawless
```

When all 8 spines are shelved together, the titles read:

```
SEE | HEAL | STAND | LIVE | GIVE | SERVE | THRIVE | BECOME
```

**Spine text rules:**
- Text reads top-to-bottom (standard US convention)
- Minimum 0.0625" clearance from spine edges
- Title should be the most prominent element on the spine
- Author name at the bottom of the spine

## Companion Workbook Covers (x8)

Each volume has a **companion workbook**. The workbook covers should look like the main volume covers — same background color, same typography, same layout — with a clear visual marker denoting it as a companion workbook.

```
│                                         │
│    ── Companion Workbook ──             │  ← Workbook designation (white, small,
│                                         │     with decorative rules or lighter weight)
│                                         │
│    Jennifer Brooke Lawless              │  ← Author
│                                         │
└─────────────────────────────────────────┘
```

### Workbook Design Notes

- **Identical design to the main volume** — same background color, title treatment, layout structure
- **"Companion Workbook" designation** — placed below the subtitle, clearly visible but secondary to the title
- Consider lighter font weight, smaller size, or decorative rules above/below "Companion Workbook"
- When shelved together (main volume + workbook side by side), they should look like a **matched pair**
- The spine should also include "Companion Workbook" or "Workbook" after the title
- Same author name: **Jennifer Brooke Lawless**

## Design Principles & Series Cohesion

### Required Consistency Across All 16 Covers (8 main + 8 workbooks)

- Same font family for titles
- Same font family for subtitles
- **All 8 titles must be the same size** — equal visual weight across every cover, regardless of character count
- White titles, off-white/warm white subtitles and secondary text
- Consistent layout structure (title position, author placement, series line position)
- "The Sovereignty Series" + volume number on each
- Single-word title as the dominant visual element — this is the brand
- The 8 spines must look unified when shelved together

### Design Principles

- **Simple over busy** — the single-word titles are the brand; let them breathe
- **Readable at thumbnail** — most book browsing happens on mobile
- **Genre-appropriate** — self-help / personal transformation, not clinical psychology
- **No stock photos with people** — tends to look cheap at this genre level
- **No borders or frames** — can look dated
- **Cohesive on a shelf** — all volumes + workbooks should feel like one unified set

### Paperback Technical Requirements

- Format: PDF/X-1a or high-quality print PDF
- Resolution: 300 DPI minimum
- Color mode: CMYK
- Bleed: 0.125" on all sides
- Safe zone: 0.25" from trim edges (0.5" for important content)
- Embed all fonts
- No crop marks (Amazon adds these)

### Ebook Technical Notes

- Title must be readable at 80px height (thumbnail browsing on mobile)
- Avoid text near edges — Amazon adds a slight border
- Ebook cover should match the front panel of the paperback wrap exactly (just RGB instead of CMYK)
