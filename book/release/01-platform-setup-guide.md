## 1. Amazon KDP (Kindle Direct Publishing)

**Purpose:** Reach, discoverability, credibility, reviews

### Account Setup

1. Go to [kdp.amazon.com](https://kdp.amazon.com)
2. Sign in with existing Amazon account or create new one
3. Complete tax interview (have SSN/EIN ready)
4. Add bank account for royalty payments

### Key Settings

- **Royalty Option:** Choose 70% for ebooks priced $2.99-$9.99
- **KDP Select:** Consider enrolling for first 90 days (exclusive to Amazon but gets Kindle Unlimited reads)
- **Pre-order:** Available up to 90 days before release

### What You'll Need

**For Kindle eBook:**
- [ ] Book manuscript (DOCX or EPUB format)
- [ ] Book cover (2560 x 1600 pixels)
- [ ] Book description (up to 4,000 characters â€” see `03-book-descriptions.md`)
- [ ] Author bio (see `07-author-bio-versions.md`)
- [ ] 7 keyword boxes (see `03-book-descriptions.md` for optimized strategy)
- [ ] Categories (choose 2)
- [ ] Series name and number

**For Paperback (publish this first):**
- [ ] Print-ready PDF (6" x 9" interior, see `04-formatting-guide.md`)
- [ ] Full cover with spine (use KDP Cover Calculator for spine width)
- [ ] ISBN (use Amazon's free ISBN for Vol 1)
- [ ] Same description, keywords, and categories as ebook

### Recommended Categories for Vol 1

- Self-Help > Abuse
- Self-Help > Relationships
- Psychology > Personality > Narcissism

### Recommended Categories for Vol 2

- Self-Help > Personal Growth > Self-Esteem
- Psychology > Psychopathology > Post-Traumatic Stress Disorder
- Self-Help > Relationships

## 2. Gumroad

**Purpose:** Direct sales, email capture, higher margins (90%+)

### Account Setup

1. Go to [gumroad.com](https://gumroad.com)
2. Create account with email
3. Connect Stripe for payments
4. Set up profile with author bio

### Product Setup

1. Click "New Product"
2. Upload PDF of book
3. Add product description
4. Set price
5. Enable "Collect emails" (CRITICAL)
6. Add cover image

### Key Features to Enable

- [ ] Email collection (automatic)
- [ ] Affiliate program (optional, 10-20% commission)
- [ ] Discount codes (for launch promotions)
- [ ] Membership tiers (for future offerings)

### Pricing Strategy

## 3. Substack

**Purpose:** Newsletter, community, content hub, email list ownership

### Account Setup

1. Go to [substack.com](https://substack.com)
2. Create account
3. Choose publication name (consider: your name, or thematic like "Heal Newsletter")
4. Write "About" page

### Critical Setup Steps

1. **Welcome Email:** Write automated email for new subscribers
2. **About Page:** Explain what readers will get
3. **Pinned Post:** Create a "Start Here" post linking to key resources
4. **Notes:** Enable Substack Notes for short-form content

### Recommended Structure

```
Publication Name: [Your Name] or "From Survival to Sovereignty"

About: Weekly insights on recognizing manipulation, healing attachment 
wounds, and building relationships that regulate rather than activate.

Posting Schedule: Weekly (same day each week)
```

### Free vs Paid

Start with **free only** to build audience. Add paid tier later when you have:
- 1,000+ free subscribers
- Consistent engagement
- Additional content to offer (deep dives, Q&A, community)

### Integration Points

- Add Gumroad links in welcome email
- Cross-promote books in every 3rd-4th newsletter
- Use Substack Notes for daily/quick content
